# 🌊 راهنمای دسکتاپ Wayland، کاشی‌کار Niri و ابزارهای Fuzzel (`Niri Desktop & Wayland Guide`)

کاشی‌کار **Niri** یک مدیر پنجره مدرن، روان و مبتنی بر Wayland است که پنجره‌ها را به جای کاشی‌کاری سنتی شبکه‌ای، در **ستون‌های افقی بی‌نهایت (Scrollable Tiling Columns)** می‌چیند. این طراحی، همراه با لانچر Fuzzel، تجربه‌ای ۱۰۰٪ کیبورد‌محور و بدون نیاز به موس را خلق می‌کند.

---

## 🪟 ۱. مدیریت پنجره‌ها، ستون‌ها و تب‌ها (`Window & Column Management`)

در Niri، هر پنجره یا به صورت یک ستون مستقل است، یا با پنجره‌های دیگر درون یک ستون ادغام شده است:

### 🎮 جدول میانبرهای اصلی ناوبری و مدیریت پنجره‌ها

| کلید میانبر | دستور Niri | توضیحات و کاربرد |
| :--- | :--- | :--- |
| **`Mod + Return`** | `spawn "kitty"` | باز کردن ترمینال اصلی (`Kitty`) |
| **`Mod + Slash`** | `spawn "kitty" --class quick-term` | باز کردن ترمینال سریع شناور (`Quick Term`) |
| **`Mod + B`** | `spawn "zen-beta"` | باز کردن مرورگر پیش‌فرض (`Zen Browser`) |
| **`Mod + E`** | `spawn "thunar"` | باز کردن فایل‌منیجر گرافیکی (`Thunar`) |
| **`Mod + Shift + E`** | `spawn "kitty" -e yazi` | باز کردن فایل‌منیجر ترمینالی `Yazi` |
| **`Mod + Q`** | `close-window` | بستن پنجره فعال جاری |
| **`Mod + Shift + Q`** | `quit` | خروج کامل از نشست Niri و دسکتاپ |
| **`Mod + T`** | `toggle-window-floating` | شناور (Floating) یا کاشی کردن پنجره جاری |
| **`Mod + F`** | `maximize-column` | حداکثر کردن عرض ستون فعلی در صفحه |
| **`Mod + Shift + F`** | `fullscreen-window` | تمام‌صفحه کردن پنجره (حذف حاشیه‌ها و بار) |
| **`Mod + C`** | `center-column` | متمرکز و وسط‌چین کردن ستون جاری در مانیتور |
| **`Mod + H / L`** | `focus-column-left / right` | حرکت نشانگر به ستون سمت چپ یا راست |
| **`Mod + K / J`** | `focus-window-up / down` | حرکت نشانگر به پنجره بالا یا پایین در ستون جاری |
| **`Mod + Ctrl + H / L`**| `move-column-left / right` | جابجا کردن کل ستون جاری به چپ یا راست |
| **`Mod + 1 تا 5`** | `focus-workspace 1..5` | پرش سریع به ورک‌سپیس ۱ تا ۵ |
| **`Mod + Shift + 1..5`**| `move-column-to-workspace 1..5`| پرتاب کردن ستون جاری به ورک‌سپیس ۱ تا ۵ |
| **`Mod + Tab`** | `toggle-overview` | باز کردن نمای کلان (`Overview`) برای جستجو و مدیریت ورک‌سپیس‌ها |

## 📑 ۲. حالت «پنجره در پنجره» و تب‌های ستونی (`Consume, Expel & Tabs`)

یکی از شاهکارهای Niri این است که شما می‌توانید پنجره‌ها را دقیقاً مثل تب‌های مرورگر درون یک ستون واحد سوار کنید:

* **مکیدن پنجره به درون ستون (`Mod + W` -> `consume-window-into-column`):**
  اگر ترمینال سمت چپ و مرورگر سمت راست است، روی ترمینال **`Mod + W`** را بزنید. ترمینال به داخل ستون مرورگر مکیده می‌شود و هر دو روی هم قرار می‌گیرند!
* **تغییر حالت نمایش به تب‌های عمودی (`Mod + G` -> `toggle-column-tabbed-display`):**
  با زدن این کلید، پنجره‌های داخل ستون به صورت تب‌های شیک بالای ستون نمایش داده می‌شوند و با `Mod + J/K` بین تب‌ها حرکت می‌کنید.
* **پرتاب کردن پنجره به بیرون از ستون (`Mod + Shift + W` -> `expel-window-from-column`):**
  پنجره فعال از ستون جدا شده و دوباره به یک ستون مستقل تبدیل می‌شود.

## 📏 ۳. عرض‌های از پیش‌تعیین‌شده و چرخش سریع (`Preset Column Widths`)

به لطف تنظیمات انجام‌شده در `config.kdl`، ستون‌ها دارای ۴ عرض استاندارد هستند:
* `۳۳.۳٪` (یک‌سوم صفحه - عالی برای چت یا ترمینال کمکی)
* `۵۰.۰٪` (نصف صفحه - استاندارد کدنویسی)
* `۶۶.۷٪` (دو‌سوم صفحه - عالی برای مرورگر وب)
* `۱۰۰.۰٪` (تمام‌صفحه)

> **✨ چرخش سریع با `Mod + R` (`switch-preset-column-width`):** هر بار که روی هر پنجره‌ای کلید **`Mod + R`** را می‌زنید، عرض پنجره به صورت نرم و انیمیشنی به عرض استاندارد بعدی جهش می‌کند. همچنین با **`Mod + Minus`** و **`Mod + Equal`** می‌توانید عرض ستون را ۱۰٪ کم یا زیاد کنید.

## 🌊 ۴. انیمیشن‌ها و فیزیک فنری (`Spring Animations`)

در کانفیگ Niri، انیمیشن‌های باز شدن، بسته شدن، تغییر سایز و حرکت بین ورک‌سپیس‌ها بر پایه **فیزیک فنری (`Spring Physics: damping-ratio=0.85, stiffness=800..1000`)** تنظیم شده‌اند. این پارامترها باعث می‌شوند پنجره‌ها حسی بسیار زنده، سریع و فنری داشته باشند بدون اینکه تاخیری در کار کیبورد ایجاد شود.

## 🚀 ۵. مرکز فرماندهی Fuzzel (`Fuzzel Superpowers`)

لانچر Fuzzel با تم Catppuccin Mocha فقط بازکننده برنامه نیست؛ بلکه به ۴ ابزار کیبورد‌محور زیر مجهز است:

### ۱. اتصال سریع به وای‌فای (`Mod + N` -> `fuzzel-wifi`)
با زدن **`Mod + N`**، منوی Fuzzel باز شده و تمام وای‌فای‌های اطراف با قدرت سیگنال را نشان می‌دهد. با جهت‌نما شبکه را انتخاب کنید، اگر رمز خواست تایپ کنید و متصل شوید (بدون هیچ کلیک موس).

### ۲. ماشین‌حساب زنده و کپی آنی (`Mod + Alt + C` -> `fuzzel-calc`)
با زدن **`Mod + Alt + C`**، یک پرامپت `🧮 Calc >` باز می‌شود. عباراتی مثل `125 * 43`، `100 USD to EUR` یا `sin(45)` را تایپ کنید و Enter بزنید؛ پاسخ آنی محاسبه شده و **مستقیماً در کلیپ‌بورد کپی می‌شود!**

### ۳. مدیریت و جستجوی تاریخچه کلیپ‌بورد (`Mod + V` -> `cliphist`)
با زدن **`Mod + V`**، لیست تمام متن‌ها و کپی‌های اخیر شما در Fuzzel باز می‌شود. روی هر کدام Enter بزنید، به کلیپ‌بورد بازمی‌گردد و آماده پیست (`Ctrl+Shift+V`) است.
*(برای پاکسازی کل دیتابیس کلیپ‌بورد در ترمینال: `cliphist wipe && wl-copy --clear`)*

## 📸 ۶. اسکرین‌شات، ضبط صفحه و نمایش کلیدها (`Screenshots & Recording`)

تمام نیازهای مستندسازی و ضبط شما روی کلیدهای سریع زیر مپ شده‌اند:

| میانبر | دستور یا اسکریپت | عملکرد |
| :--- | :--- | :--- |
| **`Mod + P`** | `screenshot` | گرفتن عکس سریع از کل مانیتور و ذخیره در دیسک |
| **`Mod + Shift + P`** | `screenshot-screen` | گرفتن عکس از مانیتور فعال جاری |
| **`Mod + Alt + P`** | `screenshot-window` | گرفتن عکس دقیق فقط از پنجره فعال |
| **`Mod + Ctrl + P`** | `capture-region-clipboard` | **انتخاب تعاملی ناحیه صفحه و کپی مستقیم در کلیپ‌بورد** (بدون ذخیره در دیسک) |
| **`Mod + Alt + R`** | `record-screen` (`gpu-screen-recorder-gtk`) | **باز کردن منوی گرافیکی ضبط صفحه** برای انتخاب ناحیه/پنجره و قطع/وصل صدا. اگر ضبط در حال اجرا باشد، با زدن مجدد این کلید ضبط متوقف شده و در `~/Videos/` ذخیره می‌شود. |
| **`Mod + Alt + K`** | `toggle-keycaster` (`wshowkeys`) | **روشن/خاموش کردن نمایش زنده کلیدهای کیبورد روی صفحه** (بسیار کاربردی برای ضبط ویدیوهای آموزشی). |
